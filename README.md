# auto-type-confirm-text
自动输入需要二次确认的文本。

暂支持以下站点，删除项目时的输入项目名的二次确认。

## 支持站点
- [x] Github
- [x] Gitee
- [x] codeup.aliyun.com
- [x] vercel.com
- [x] dash.cloudflare.com

如需支持其它站点，可反馈，将酌情添加。
